# GitHub Repository Setup - Vaibhav Portfolio

## Plan Steps:
1. ✅ Create .gitignore for Astro/Node.js project.
2. ✅ Create README.md with project details and instructions.
3. Stage all files: `git add .`
4. Initial commit: `git commit -m "Initial commit: Vaibhav Portfolio (Astro)"`
5. Add GitHub remote: `git remote add origin https://github.com/VAIBHAVDADHICH7/vaibhav-portfolio.git`
6. Push: `git branch -M main && git push -u origin main`

## Notes:
- User needs to create empty repo 'vaibhav-portfolio' on GitHub first (no README, .gitignore, license).
- Check GitHub CLI (`gh auth status`) for easier PR/branch later if needed.
- After push, run `npm run build` to verify.
