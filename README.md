# Vaibhav Portfolio

Personal portfolio website built with **Astro**, **React**, and **Tailwind CSS**. This is my portfolio website.

## 🚀 Features
- Responsive design
- Smooth animations (Framer Motion)
- Optimized performance (Lighthouse scores)
- React components for interactive sections

## 📦 Quick Start

### Prerequisites
- Node.js (v18+)

### Install & Run
```bash
cd astro
npm install
npm run dev      # Development server: http://localhost:4321
npm run build    # Build for production
npm run preview  # Preview production build
```

### Deployment
- **Vercel/Netlify**: Connect GitHub repo and deploy automatically.
- Built assets in `astro/dist/`.

## 🛠️ Tech Stack
- Astro 5.0
- React 18
- Tailwind CSS 3.4
- Framer Motion

Live demo: [Coming soon](https://vaibhav-portfolio.vercel.app)
